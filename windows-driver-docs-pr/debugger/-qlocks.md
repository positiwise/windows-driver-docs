# !qlocks


The **!qlocks** extension displays the state of all queued spin locks.

```dbgcmd
!qlocks 
```

## Remarks

This command is useful only on a multiprocessor system.

Here is an example:

```dbgcmd

Key: O = Owner, 1-n = Wait order, blank = not owned/waiting, C = Corrupt
```
