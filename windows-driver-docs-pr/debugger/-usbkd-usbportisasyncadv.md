# !usbkd.usbportisasyncadv


The **!usbkd.usbportisasyncadv** command checks all EHCI miniport drivers for an EHCI Interrupt on Async Advance issue.

```dbgcmd
!usbkd.usbportisasyncadv
```

## <span id="DLL"></span><span id="dll"></span>DLL


Usbkd.dll
